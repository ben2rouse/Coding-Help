Variables

Variables (int, str, bool, float)

Think of a variable like a labeled mailbox that stores something for you. The label is the name, and inside is the value.

int ‚Üí whole numbers (e.g., 7, -2)
float ‚Üí decimals (e.g., 5.6, 3.14)
str ‚Üí text/strings (e.g., "hello")
bool ‚Üí True or False
# Make some mailboxes:
age = 13          # int
height = 5.6      # float
name = "Jordan"   # str
likes_pizza = True  # bool

# You can use them later:
print(name, "is", age, "years old.")
Printing

Printing in Python

Printing means showing information on the screen.

Commas vs + in print()

Commas automatically put spaces and can mix types easily.
+ joins strings only ‚Äî so you must convert numbers to strings first. 

# Using commas (easy mixing):
age = 13
print("I am", age, "years old.")  # no need to convert

# Using + (string only):
name = "Jordan"
print("Hello, my name is " + name)     # works
# print("I am " + age + " years old.") # ‚ùå error: age is an int
print("I am " + str(age) + " years old.")  # ‚úÖ convert with str()
Tip: If something isn‚Äôt a string yet (like numbers), wrap it with str(...) when using +.
Loops

Loops

Loops repeat code for you. Two kinds we use now:
for ‚Üí repeat a set number of times.
while ‚Üí keep going until a condition is False.

for loop (repeat N times)

# Count from 1 to 5
for i in range(1, 6):
    print("Counting:", i)

while loop (repeat until condition is met)

# Keep going until num reaches 5
num = 1
while num <= 5:
    print("num is", num)
    num = num + 1  # don't forget to change num!
Important: A while loop needs something in the loop to eventually make the condition False, or it will run forever.
Types

Type Conversion (changing types)

Sometimes you need to change a value‚Äôs type. Use these:

str(x) ‚Üí make text
int(x) ‚Üí make an int (whole number)
float(x) ‚Üí make a decimal
bool(x) ‚Üí make True/False
age_text = "13"
age_number = int(age_text)   # "13" ‚Üí 13

height = 5.6
height_text = str(height)    # 5.6 ‚Üí "5.6"

print("Age as number:", age_number)
print("Height as text:", height_text)
Math

Math Operations

+ add
- subtract
* multiply
/ divide (gives float)
// floor divide (drops decimals)
% remainder (modulo)
** power (exponent)
a = 10
b = 3
print("a + b =", a + b)    # 13
print("a - b =", a - b)    # 7
print("a * b =", a * b)    # 30
print("a / b =", a / b)    # 3.333...
print("a // b =", a // b)  # 3
print("a % b =", a % b)    # 1
print("a ** b =", a ** b)  # 1000
Use cases: % is great for checking even/odd: n % 2 == 0 means even.

Using Random Numbers

Python‚Äôs random library lets you create random numbers for math games and challenges. You can use it for guessing games, quizzes, or generating random scores. 

import random

# Pick a random integer between 1 and 10 (inclusive)
num = random.randint(1, 10)
print("Random number:", num)

# Pick a random floating-point number between 0 and 1
decimal = random.random()
print("Random decimal:", decimal)

# Choose a random item from a list
choice = random.choice(["add", "subtract", "multiply", "divide"])
print("Random math choice:", choice)
Tip: Every time you run the program, random numbers change. Use import random once at the top of your code to start using it.
I/O

Input & Output

Input = the computer reads what you type. (It always comes in as text/str.)
Output = the computer prints information to the screen. 

# Ask for info (input ‚Üí str), then convert if needed:
name = input("What is your name? ")
age_text = input("How old are you? ")
age = int(age_text)  # now age is a number (int)

print("Hello,", name, "! Next year you will be", age + 1)

# Shortcut: convert directly
score = int(input("Enter your score: "))  # str ‚Üí int in one line
height = float(input("Enter your height: "))  # str ‚Üí float
Conditions

Conditions & Operators

Conditions let your code make decisions. We use comparison operators to check things:

== equal to
!= not equal to
> greater than
< less than
>= greater or equal
<= less or equal
score = 75

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Keep practicing!")

Logic (and / or)

Combine checks with and (both must be True) or or (at least one is True). 

age = 13
age = 12
year = 2000

if (age == 12) and (year == 2000):
    print("You are 12 in the year 2000!")
else:
    print("Condition not met.")
