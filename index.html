Python Basics – Class Notes

Simple explanations + small examples to help you practice.

Variables

Variables (int, str, bool, float)

Think of a variable like a labeled mailbox that stores something for you. The label is the name, and inside is the value.

int → whole numbers (e.g., 7, -2)
float → decimals (e.g., 5.6, 3.14)
str → text/strings (e.g., "hello")
bool → True or False
# Make some mailboxes:
age = 13          # int
height = 5.6      # float
name = "Jordan"   # str
likes_pizza = True  # bool

# You can use them later:
print(name, "is", age, "years old.")
Printing

Printing in Python

Printing means showing information on the screen.

Commas vs + in print()

Commas automatically put spaces and can mix types easily.
+ joins strings only — so you must convert numbers to strings first. 

# Using commas (easy mixing):
age = 13
print("I am", age, "years old.")  # no need to convert

# Using + (string only):
name = "Jordan"
print("Hello, my name is " + name)     # works
# print("I am " + age + " years old.") # ❌ error: age is an int
print("I am " + str(age) + " years old.")  # ✅ convert with str()
Tip: If something isn’t a string yet (like numbers), wrap it with str(...) when using +.
Loops

Loops

Loops repeat code for you. Two kinds we use now:
for → repeat a set number of times.
while → keep going until a condition is False.

for loop (repeat N times)

# Count from 1 to 5
for i in range(1, 6):
    print("Counting:", i)

while loop (repeat until condition is met)

# Keep going until num reaches 5
num = 1
while num <= 5:
    print("num is", num)
    num = num + 1  # don't forget to change num!
Important: A while loop needs something in the loop to eventually make the condition False, or it will run forever.
Types

Type Conversion (changing types)

Sometimes you need to change a value’s type. Use these:

str(x) → make text
int(x) → make an int (whole number)
float(x) → make a decimal
bool(x) → make True/False
age_text = "13"
age_number = int(age_text)   # "13" → 13

height = 5.6
height_text = str(height)    # 5.6 → "5.6"

print("Age as number:", age_number)
print("Height as text:", height_text)
Math

Math Operations

+ add
- subtract
* multiply
/ divide (gives float)
// floor divide (drops decimals)
% remainder (modulo)
** power (exponent)
a = 10
b = 3
print("a + b =", a + b)    # 13
print("a - b =", a - b)    # 7
print("a * b =", a * b)    # 30
print("a / b =", a / b)    # 3.333...
print("a // b =", a // b)  # 3
print("a % b =", a % b)    # 1
print("a ** b =", a ** b)  # 1000
Use cases: % is great for checking even/odd: n % 2 == 0 means even.
I/O

Input & Output

Input = the computer reads what you type. (It always comes in as text/str.)
Output = the computer prints information to the screen. 

# Ask for info (input → str), then convert if needed:
name = input("What is your name? ")
age_text = input("How old are you? ")
age = int(age_text)  # now age is a number (int)

print("Hello,", name, "! Next year you will be", age + 1)

# Shortcut: convert directly
score = int(input("Enter your score: "))  # str → int in one line
height = float(input("Enter your height: "))  # str → float
Conditions

Conditions & Operators

Conditions let your code make decisions. We use comparison operators to check things:

== equal to
!= not equal to
> greater than
< less than
>= greater or equal
<= less or equal
score = 75

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Keep practicing!")

Logic (and / or)

Combine checks with and (both must be True) or or (at least one is True). 

age = 13
age = 12
year = 2000

if (age == 12) and (year == 2000):
    print("You are 12 in the year 2000!")
else:
    print("Condition not met.")
