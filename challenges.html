1) Tip Calculator

Goal: Ask for a bill total and a tip percent. Calculate the tip and the final total. Print both nicely.

  • Get two inputs: bill (like "42.50") and tip percent (like "18").
  • Convert to numbers with float() or int().
  • Compute: tip = bill * (percent/100), total = bill + tip.
  • Print the tip and total with labels.

2) Even or Odd Checker

Goal: Ask for a number and tell the user if it’s even or odd.

  • Input a number → convert to int.
  • Use the modulo operator % to find the remainder when dividing by 2.
  • If n % 2 == 0 → “even”, else “odd”.
Reminder: a % b gives the remainder. Example: 7 % 2 is 1.

See: Math • Conditions

  • Use a for loop with range(1, 11).
  • Print the message each time through the loop.

5) Simple Guessing Game

Goal: Pick a secret number in your code. Keep asking the user to guess until they get it right.

  • Make secret = 7 (or any number).
  • Use a while loop that runs while the guess is not equal to the secret.
  • Tell the user “Too high” / “Too low” each time.
How while works: It keeps repeating while the condition is True. Change something inside the loop so it can eventually stop.

See: Loops • Conditions • Input

6) Countdown Timer

Goal: Ask the user for a start number and count down to 1, then print “Blast off!”.

  • Convert the input to int.
  • Use range(start, 0, -1) to go down by 1.
  • Print each number; after the loop, print “Blast off!”.
range(start, stop, step): stop is not included. Use a negative step (like -1) to count backwards.

See: Loops

7) Simple Bank Balance

Goal: Start with a balance (e.g., 100). Show a menu to check, deposit, withdraw, or exit. Repeat until exit.

  • Use a while loop to keep showing the menu.
  • 1 → print balance, 2 → add amount, 3 → subtract amount (if enough money), 4 → break the loop.
# Mini pattern:
running = True
while running:
    # print menu and read choice
    # use if/elif/else to handle actions
    # set running = False to exit
