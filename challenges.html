1) Tip Calculator

Goal: Ask for a bill total and a tip percent. Calculate the tip and the final total. Print both nicely.

  • What to build: A short program that reads a bill amount and a tip percent, then prints the tip and the grand total.
  • Inputs:
    • bill: a positive number, may include cents (e.g., 42.50).
    • percent: a whole number from 0–100 (e.g., 18 for 18%).
  • Process:
    • Convert inputs using float() (for bill) and int() (for percent).
    • Compute tip = bill * (percent / 100).
    • Compute total = bill + tip.
    • Round both tip and total to exactly 2 decimal places when printing.
  • Output (exact labels):
    • Tip: $X.XX
    • Total: $Y.YY
  • Rules: Don’t include the $ in inputs. If the percent is 0, the tip is $0.00.
  • Example run: 
    Bill amount: 42.50
Tip percent: 18
Tip: $7.65
Total: $50.15

2) Even or Odd Checker

Goal: Ask for a number and tell the user if it’s even or odd.

  • Input: one integer (e.g., 7). Convert with int().
  • Logic: Use modulo % with 2.
    • If n % 2 == 0 → print "even".
    • Else → print "odd".
  • Output: print only the word "even" or "odd" (all lowercase), nothing else.
  • Edge cases: 0 is even. Negative numbers follow the same rule.
  • Example: 
    Number: 14
even
Reminder: a % b gives the remainder. Example: 7 % 2 is 1.

4) Repeat My Sentence

Goal: Ask the user for a short sentence and how many times to repeat it. Then print it that many times.

  • Inputs:
    • sentence: any non-empty string.
    • count: an integer >= 0.
  • Loop: run count times using range(count).

  • Edge cases: if count is 0, print nothing.
  • Example:

5) Simple Guessing Game

Goal: Pick a secret number in your code. Keep asking the user to guess until they get it right.

  • Setup: Choose a secret integer in your code (e.g., secret = 7).
  • Loop: Use a while loop. Ask for a guess each time. Convert it with int().
  • Feedback:
    • If guess < secret → print "Too low".
    • If guess > secret → print "Too high".
  • Win message: when correct, print "You got it in X tries." where X is the number of guesses.
  • Example transcript: 
    Guess a number: 3
Too low
Guess a number: 9
Too high
Guess a number: 7
You got it in 3 tries.
How while works: It keeps repeating while the condition is True. Change something inside the loop so it can eventually stop.

6) Countdown Timer

Goal: Ask the user for a start number and count down to 1, then print “Blast off!”.

  • Input: start as an integer.
  • Loop: use range(start, 0, -1) to go down by 1.
  • Output: print each number on its own line; after the loop print Blast off!.
  • Validation (simple): if start < 1, print Enter a number 1 or greater. and stop.
  • Example: 
    Start at: 5
5
4
3
2
1
Blast off!
range(start, stop, step): stop is not included. Use a negative step (like -1) to count backwards.

  • Start: balance = 100.00 (float).
  • Menu (repeat until exit):

3) Withdraw
4) Exit

  • Sample session:

2) Deposit
3) Withdraw
4) Exit

Balance: $100.00
Choose: 2
Amount: 25.5

Goodbye!

10) Number Pattern
